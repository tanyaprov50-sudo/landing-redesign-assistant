# Landing Redesign Assistant

Агент для анализа лендингов и получения рекомендаций по улучшению дизайна и структуры с помощью OpenAI.

## Описание

Программа принимает URL лендинга, загружает HTML страницы, извлекает текст и отправляет его в OpenAI API для получения 5 конкретных рекомендаций по улучшению:
- Структуры страницы
- Заголовков
- Призывов к действию (CTA)
- Визуальной иерархии
- Доверия пользователя

Все отчёты автоматически сохраняются в директорию `reports/`.

## Требования

- Python ≥ 3.10
- OpenAI API ключ (поддерживается ProxyAPI)

## Установка

1. Клонируйте или скачайте проект:
```bash
cd landing_redesign_assistant
```

2. Создайте виртуальное окружение (рекомендуется):
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Настройте переменные окружения:
```bash
cp .env.example .env
```

5. Откройте файл `.env` и добавьте ваш OpenAI API ключ:
```
OPENAI_API_KEY=sk-your-actual-api-key-here
```

## Использование

### Способ 1: Через bat-файл (Windows)
1. Откройте папку `landing_redesign_assistant` в проводнике
2. Двойной клик на файл `run.bat`
3. Программа запустится автоматически

### Способ 2: Через командную строку
```bash
cd landing_redesign_assistant
python agent.py
```

Введите URL лендинга, когда программа попросит:
```
Введите URL лендинга: https://example.com
```

Программа:
1. Загрузит HTML страницы
2. Извлечёт текст (удалив script, style, nav, footer)
3. Отправит текст в OpenAI
4. Выведет 5 рекомендаций в консоль
5. Сохранит отчёт в `reports/redesign_report_YYYYMMDD_HHMMSS.txt`

## Получение OpenAI API ключа

### Вариант 1: Через ProxyAPI (для России)
1. Зарегистрируйтесь на [proxyapi.ru](https://proxyapi.ru)
2. Получите API ключ
3. Скопируйте ключ в файл `.env`

### Вариант 2: Напрямую через OpenAI
1. Зарегистрируйтесь на [platform.openai.com](https://platform.openai.com)
2. Перейдите в раздел API Keys
3. Создайте новый ключ
4. Скопируйте ключ в файл `.env`
5. В `openai_module.py` удалите параметр `base_url` из инициализации клиента
